**bin/install.mjs** -- CLI entry point (shebang: `#!/usr/bin/env node`):
- Parse `process.argv[2]` as the command. Only `install` is valid. If missing or unrecognized, print usage and exit 1.
- Support optional `--runtime <name>` flag to skip auto-detection (values: `claude-code`, `opencode`, `gemini`, `codex`).
- Flow:
  1. Call `checkDjinnServer()` from preflight.mjs. If false, print error: "Error: djinn-server is not installed or not on PATH.\nDownload Djinn Desktop from https://djinnai.io or install djinn-server manually." Exit 1.
  2. If `--runtime` specified, install for that runtime only.
  3. Otherwise, call `detectRuntimes()`. If 0 found, print error and exit. If 1 found, auto-install for it. If 2+ found, prompt user to select which runtimes to install for (using readline for interactive selection -- display numbered list, accept comma-separated numbers).
  4. For each selected runtime, call the corresponding installer (imported dynamically from `lib/install-{runtime}.mjs`).
  5. Print summary of what was installed and where.

**lib/preflight.mjs** -- Export `checkDjinnServer()`:
- Use `child_process.execSync('which djinn-server', { stdio: 'pipe' })` wrapped in try/catch. Return true if exit code 0, false otherwise.
- On Windows, use `where djinn-server` instead. Check `process.platform === 'win32'`.

**lib/detect.mjs** -- Export `detectRuntimes()`:
- Check for each runtime's config directory existence using `fs.existsSync()`:
  - `claude-code`: `~/.claude/` exists
  - `opencode`: `~/.config/opencode/` exists
  - `gemini`: `~/.gemini/` exists
  - `codex`: `~/.codex/` exists
- Use `os.homedir()` for `~` expansion.
- Return array of `{ name, configDir }` objects for detected runtimes.

**JSON merge helper** (shared within each JSON installer):
```javascript
function mergeJsonConfig(configPath, keyPath, value) {
  let config = {};
  if (existsSync(configPath)) {
    config = JSON.parse(readFileSync(configPath, 'utf-8'));
  }
  const parts = keyPath.split('.');
  let current = config;
  for (let i = 0; i < parts.length - 1; i++) {
    current[parts[i]] = current[parts[i]] || {};
    current = current[parts[i]];
  }
  current[parts[parts.length - 1]] = value;
  writeFileSync(configPath, JSON.stringify(config, null, 2) + '\n');
}
```
